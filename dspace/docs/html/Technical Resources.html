Technical Resources

dspace.osuosl.org

is currently hosted at OSL and accessible by DSpace project administrators upon request.

scm.dspace.org

vhost on dspace.osuosl.org

maven.dspace.org

This vhost is used for managing both snapshot and release builds of DSpace artifacts. The release directory is rsynced with the maven central repository on a 3hr cycle.

maven.dspace.org is identical to scm.dspace.org

projects.dspace.org

This vhost is used for managing the publication of maven project documentation.

projects.dspace.org is identical to scm.dspace.org

libaxis1.mit.edu

Is a webhosting service maintained by MIT. Access to this machine is managed by Alex Brennan at MIT.

Previously maven.dspace.org and projects.dspace.org were both hosted here. But, they have been migrated to OSUOSL as of Nov 2009.
