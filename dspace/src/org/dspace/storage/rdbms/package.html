Provides an API for accessing a relational database management system. The main class is {@link org.dspace.storage.rdbms.DatabaseManager}, which executes SQL queries and returns {@link org.dspace.storage.rdbms.TableRow} or {@link org.dspace.storage.rdbms.TableRowIterator} objects. The {@link org.dspace.storage.rdbms.InitializeDatabase} class is used to load SQL into the RDBMS via JDBC. The {@link org.dspace.storage.rdbms.SimplePool} class provides a database connection pooling API.
